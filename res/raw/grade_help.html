Noten Hilfe

Neue Note:

Wenn du eine neue Noten eintragen willst, drücke auf den Menu Button deines Gerätes und drücke auf "Neu".

Note ändern:

Um eine Note zu ändern, drücke auf die Note in der Liste.

Note löschen:

Drücke in der Notenliste etwas länger auf eine einzelne Note. Es öffnet sich ein Menu mit einem Button "löschen".
